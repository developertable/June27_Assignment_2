## üöÄ Getting Started

To run the project locally:

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/taskify.git
    cd taskify
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the development server:
    ```bash
    npm start
    ```

4. Open your browser at `http://localhost:3000`
